# Genet

Genet is an open-source curated collection of useful tools for developers and designers.

The goal of Genet is to make it easier to discover websites, resources, and services that help build better products on the web. From UI/UX tools to development utilities, Genet organizes modern web tools in one place.

---

## Overview

Genet is a simple and minimal platform that presents useful tools as organized cards with metadata such as:

- tool name
- logo
- short description
- category
- tags
- link to the website

The project is open source and designed to grow with community contributions.

---

## Features

- Curated collection of developer and design tools
- Minimal and fast interface
- Tool categorization
- Tag-based filtering
- Easy contribution system
- Static and lightweight architecture
